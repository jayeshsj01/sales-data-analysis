# Sales Data Analysis Project

## Overview
This project focuses on analyzing sales data to understand business performance and identify key trends. The goal was to work with real-world data, clean it, and generate meaningful insights using Python.

## What I Did
- Loaded and cleaned raw sales data using Pandas  
- Handled data type conversions and missing values  
- Performed analysis on sales trends, products, and regions  
- Created visualizations using Matplotlib  
- Improved graph readability by adding labels, markers, and highlights  
- Saved all generated graphs for reporting purposes  

## Key Insights
- Sales peaked in November, indicating strong seasonal demand  
- Certain product lines (like Classic Cars) contributed the highest revenue  
- The USA was the top-performing market  
- Medium-sized deals generated the most revenue  

## Tools & Technologies
- Python  
- Pandas  
- Matplotlib
