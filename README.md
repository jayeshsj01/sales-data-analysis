# 📊 Sales Data Analysis Dashboard

## Overview

This project is based on analyzing a sales dataset to understand business performance and extract useful insights. I worked on this project to improve my skills in data analysis using Python, SQL, and Power BI.

## What I Did

* Cleaned and explored the dataset using Python (Pandas)
* Handled data issues like formatting and missing values
* Wrote SQL queries to analyze sales trends and patterns
* Built an interactive dashboard in Power BI to visualize insights
* Added filters (slicers) to make the dashboard dynamic and easy to use

## Dashboard Features

* Total Revenue, Average Order Value, and Total Orders (KPI cards)
* Monthly sales trend to identify seasonality
* Product line performance comparison
* Country-wise sales analysis
* Deal size distribution using a pie chart

## Key Insights

* Sales were highest towards the end of the year, showing seasonal trends
* Classic Cars generated the most revenue among all product lines
* USA contributed the highest share of total sales
* Large deal sizes had the biggest impact on overall revenue
